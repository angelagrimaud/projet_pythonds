# Projet python pour la data science : étude de la variation de la consommation d'électricité en fonction de la météo

L'objectif de ce projet, réalisé dans le cadre du cours de l'ENSAE "Python pour la datascience", est d'analyser les variations de la consommation d'électricité, et plus particulièrement d'essayer d'expliquer ces variations par la météo.

Nous nous appuierons principalement sur les bases de données "Consommation d'électricité des grandes Métropoles françaises temps réel" de RTE et publié sur Open Data-Réseau Énergie (ODRÉ) et "Données climatologiques de base – horaires" de MétéoFrance, publiées sur le site data.gouv.
